# ادعيله لرونالدو — Dua for CR7

صفحة تريبيوت من الجمهور لكريستيانو رونالدو في مونديال 2026، فيها عداد دعوات وحيطة رسايل، وثنائية اللغة (عربي/إنجليزي).

## تشغيل الموقع
الصفحة عبارة عن ملف HTML واحد (`index.html`) + فولدر `images/`. تقدر تفتحها مباشرة من جهازك، أو تنشرها على GitHub Pages:

1. اعمل push للفولدر ده على ريبو جديد على GitHub.
2. من إعدادات الريبو (Settings → Pages) اختار الفرع اللي فيه الكود (زي `main`) والفولدر `/ (root)`.
3. GitHub هيديك لينك زي `https://username.github.io/repo-name/`.

## الداتا بيس
العداد وحيطة الرسايل بيتخزنوا في `localStorage` بتاع كل متصفح — يعني بيفضلوا محفوظين عند نفس الزائر لما يرجع تاني، لكنهم **مش متزامنين بين الزوار المختلفين**. عشان يبقى فيه عداد ورسايل مشتركة فعلاً لكل الزوار، محتاج backend حقيقي (زي Firebase أو Supabase)، وده أعقد من نطاق الصفحة دي.

## الصور
كل الصور اللي في فولدر `images/` رسمات أصلية (SVG) اتعملت خصيصًا للصفحة دي، مش صور حقيقية لرونالدو — الموقع بيوضح كده في الفوتر كمان. لو حابب تستبدلها بصور حقيقية عندك حقوق نشرها، سيب نفس أسماء الملفات:

- `images/celebration.svg`
- `images/trophy.svg`
- `images/crest.svg`
- `images/six-world-cups.svg`

## ملاحظة
الصفحة دي مش تابعة لكريستيانو رونالدو ولا الاتحاد البرتغالي ولا الفيفا.
